需要[[选择时钟源]]
这个时基配置完了会立刻更新中断一次
测PWM要把Period拉满防止溢出
```
/*配置时基单元*/
	TIM_TimeBaseInitTypeDef TIM_TimeBaseInitStucture;
	TIM_TimeBaseInitStucture.TIM_ClockDivision = TIM_CKD_DIV1;
	TIM_TimeBaseInitStucture.TIM_CounterMode = TIM_CounterMode_Up;
	TIM_TimeBaseInitStucture.TIM_Period = 10000 - 1;
	TIM_TimeBaseInitStucture.TIM_Prescaler = 7200 - 1;
	TIM_TimeBaseInitStucture.TIM_RepetitionCounter = 0;
	TIM_TimeBaseInit(TIM2, &TIM_TimeBaseInitStucture);
	
	TIM_ClearFlag(TIM2, TIM_FLAG_Update);    //避免立刻更新中断
```

时基单元可以完成定时中断，包含
	PSC预分频器，平均分输入频率，份数从0开始，16位
	CNT计数器，对分频后的时钟计数，每上升沿加1，16位
	ARR自动重载寄存器，存储自动重装值，16位，计数器到达自动重装值时产生更新中断和更新事件并清零计数器

CK_PSC输入时钟
CK_CET输出时钟
计数器溢出频率：CK_CNT_OV = CK_CNT / (ARR + 1) = CK_PSC / (PSC + 1) / (ARR + 1)
